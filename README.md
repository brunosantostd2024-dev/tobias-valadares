# Tobias & Valadares Advogadas

Site oficial do escritório **Patrícia Tobias Santos & Maria Eduarda Valadares**.

## Funcionalidades

- Site público institucional
- Sistema administrativo completo (login: `funcionario@escritorio.com` / senha: `tobias2025`)
- Cadastro e gerenciamento de clientes
- Controle de processos
- Agenda
- Calculadora INSS com cálculo de tempo de contribuição
- Estimativa de benefícios previdenciários

## Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Node.js + Express (servidor)

## Como rodar localmente

```bash
npm install
npm start
```

Acesse: http://localhost:3000
